## 핵심기능
* 유튜브 채널 CRUD
: youtube 채널 url 입력 시 영상의 썸네일을 웹스크랩팅합니다. 
: JWT 를 이용하여 사용자 이름를 가져와 붙입니다.
* 좋아요 기능
: JWT 를 이용하여 사용자를 확인하여 좋아요 수를 카운트하고 변경합니다.
* Keyword 카테고리별 순위
: 영상 등록 시 카테고리 수를 카운트하여 순위를 목록화하여 보여줍니다. 
* Keyword 카테고리별 CRUD
: 카테고리별로 분류하여 조회가 가능합니다.
